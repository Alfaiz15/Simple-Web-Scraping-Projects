# Scrapethissite.com – World Countries Web Scraper  
**Web Scraping Pertama Saya yang 100% Berhasil & Production-Ready**

### Apa yang dilakukan project ini?
Mengambil data semua negara di dunia (nama negara, ibu kota, jumlah penduduk) dari  
https://www.scrapethissite.com/pages/simple/  
→ langsung disimpan ke file CSV dalam hitungan detik.

### Teknologi yang dipakai
- Python
- `requests`
- `BeautifulSoup4`

### Cara menjalankan (gampang banget!)
```bash
pip install requests beautifulsoup4
python scraper.py
